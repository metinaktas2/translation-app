# 🌍 Transly+

A modern and fully responsive translation application that allows users to translate text across 137 languages with an intuitive and user-friendly interface.

# 🚀 Features
- 🌐 Translate text between 137 languages
- 🔍 Auto-detect source language
- 📝 No need to manually select the source language
- 📜 View previously translated texts
- ⚡ Fast and seamless translation experience
- 📱 Fully responsive design
- 🎯 Clean and user-friendly UI
- 🛠️ Tech Stack
- React
- Redux Toolkit
- React Redux
- Axios
- React Select
- Tailwind CSS
- Lucide React
